# Blog With MVC

*Created This Simple Blog Website utilizing MVC to Practice MVC CRUD*

*Current Features*
- View Recent Blogs
- View All Blogs
- View Single Blog In Detail
- Create a Blog
- Save Blog in SQL Server

*Planned Features*
- Updating Blogs
- Deleting Blogs
